# Borja's Kimchi Lab

Two kimchi recipes. Zero pretension. Maximum fermentation.

A static recipe site featuring Classic Napa Cabbage Kimchi and Lemon Kimchi with Korean Red Chili Sauce & Charred Scallions.

## Running locally

Open `index.html` in your browser. That's it. No build step, no dependencies, no npm.

## Hosting

Drop these files on any static host or enable GitHub Pages on this repo.
